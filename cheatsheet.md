# Terminal Quick Keys

## tmux
- `C-a |` split pane horizontally
- `C-a _` split pane vertically
- `C-a h/j/k/l` move between panes
- `C-a H/J/K/L` resize panes by 5 cells
- `C-a [` enter copy mode, then `v` select, `y` copy

## vim
- `,w` save
- `,bd` close current buffer safely
- `,ss` toggle spell check
- `,rg` ripgrep project search into quickfix
- mouse click places cursor (`set mouse=a`)

## shell
- `reload` restart login shell
- `newmeeting <project> <type> "<attendees>"` scaffold transcript
- `nv` open Neovim (if installed)
- `tkeys` show this cheatsheet
